2713、矩阵中严格递增的单元格数( Hard )

m x n 的整数矩阵 mat，从 任一 起始单元格出发，你可以移动到 同一行或同一列中 的任何其他单元格，但前提是目标单元格的值 严格大于 当前单元格的值。

可多次重复这一过程，直到无法再进行任何移动。
请找出从某个单元开始访问矩阵所能访问的单元格的最大数量。

思路一：dfs图遍历
使用一个矩阵dist[m][n] 记录”当前结点开始可以访问的最长距离”
最后返回max(dist)
求解过程：逐一遍历m * n个点，向外遍历每行每列O(m + n)。
由于dist可以记录已经遍历完成的结点的结果，因此不会重复遍历
因此时间复杂度O((m * n) * (m + n))
结果正确，但是超时 (558 / 566)

思路二：
按思路一，实际上，不需要为每个结点遍历所在行列，只需要记录 __每行和每列的dist最大值__，记作row[m]和col[n], 初始为0
排序 (O(mnlog(mn))), 从小到大计算，根据{value, r, c}
得到状态转移方程

最后返回max(row)或者max(col)均可
时间复杂度O(mnlog(mn))，排序耗时最多
